4 - Build our docker image

a. Create an empty "app" folder in the root of our project.

b. Inside your project folder, run the following command to run our dockerfile:

docker build .

5 - Configure docker compose

a. Create a "docker-compose.yml" file in the root of our project.

b. Run the following command to build our image using the compose configuration:

docker-compose build

6 - Create our project

a. Run the following code to create our project using compose.

docker-compose run app sh -c "django-admin.py startproject app ."

This code will start a project in our WORKDIR that we defined in our Dockerfile.
